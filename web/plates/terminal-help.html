Terminal

The WebUI Terminal provides an interactive root shell session. You can run diagnostic and configuration commands directly from the browser without requiring SSH access.

Common commands:

  • sdrs — List connected RTL-SDR dongles
  • sdr-tool.sh — SDR EEPROM configuration utility
  • rtl_test -p — Test SDR and estimate PPM offset
  • dmesg | tail — View recent kernel messages
  • systemctl status stratux — Check service status

Session notes:

  • Sessions timeout after 15 minutes of inactivity
  • Maximum 2 concurrent terminal sessions
  • Type exit to close the session
  • Use Ctrl+C to interrupt running commands
